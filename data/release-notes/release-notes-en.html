System Requirements

  • 64-bit processor
  • At least 2 GB memory
  • At least 12 GB disk space
  • It is recomended to create swap space for Pisi Linux (can be done
    during installation)

Multimedia

Pisi Linux currently exists only in 64-bit form. However, 32-bit
packages are included for some applications.
